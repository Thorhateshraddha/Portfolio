My Projects

Online Food Ordering System

This project is a dynamic web application that allows customers to order food online from a digital menu. Users can browse available food items with images and prices, add items to their cart, and proceed to checkout.

It includes key features like order tracking, real-time price calculation, payment record management, and confirmation notifications. The admin panel helps manage the food inventory, orders, and customer details.

Technologies used include HTML5, CSS3, JavaScript for the front-end interface, and PHP & MySQL for back-end logic and data storage. The interface is fully responsive and optimized for mobile devices.

Tech Hub Admission Portal

The Tech Hub Admission Portal is a full-stack web application built to streamline the student enrollment process in a tech education environment. Students can register, select courses, and receive confirmation in a user-friendly interface.

The admin module facilitates adding new courses, managing student data, and generating reports. The application uses RESTful APIs to ensure secure communication between the client and server.

Technologies include Node.js and Express.js on the server-side, with MySQL as the database. The front-end uses HTML/CSS and JavaScript for an interactive experience. It emphasizes proper validation, modular design, and scalability.

Employee Management System

This desktop-based Java project is designed to manage employee data efficiently. It covers functionalities such as adding new employees, updating records, and deleting entries for those who leave the organization.

The system ensures that all employee records—such as name, department, role, and status—are stored and retrieved from a MySQL database. It offers a clean GUI using Java Swing and AWT, making it easy for HR personnel to use.

This system is ideal for small to mid-sized companies looking to centralize their HR data. It demonstrates core Java concepts such as OOPs, exception handling, and database connectivity using JDBC.
